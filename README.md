# 미션 - 페어매칭관리 애플리케이션

## 구현할 기능 목록 작성
### 진행과정
1. 백엔드, 프론트엔드 크루의 정보를 파일 입출력을 통해 불러온다.
2. 프로그램 기능을 선택한다.
   - 페어 매칭
   - 페어 조회
   - 페어 초기화
   - 종료

### 예외사항
1. 프로그램 기능 선택
    - 1, 2, 3, Q가 아닌 수를 입력한 경우
2. 페어 매칭 입력
    - 백엔드, 프론트엔드가 아닌 경우
    - 레벨1~5 사이가 아닌 경우
    - 미션에 속하지 않는 경우
    - 3개의 입력과 콤마(,)로 이루어지지 않은 경우
3. 페어 매칭 기능
   - 3회 시도까지 매칭되지 않은 경우
   - 매칭을 할 수 있는 경우의 수가 없는 경우

### 기능
1. 페어 매칭
    - 과정, 미션 출력
    - 매칭하고자 하는 과정, 레벨, 미션 입력
    - 매칭 성공시 결과 출력
2. 페어 조회
    - 페어 매칭된 결과 조회
3. 페어 초기화
    - 페어 정보 초기화
4. 종료
    - 프로그램 종료
